# Portfolio
Detta är min personliga portfolio som visar det jag har lärt mig från kursen design på BTH. 

## Installation
För att köra webbplatsen lokalt måste du först klona repositoryt, för att sedan köra composer install i en komandotolk eftersom koden har PHP som är i behov av ett externt biblotek.  
När det är utfört måste man starta servern och det gör man genom att köra php -S localhost:8000 i den nerladdade mappen.
